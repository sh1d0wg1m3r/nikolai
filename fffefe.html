CSS

Basic Structure

selector { - Targets HTML elements to apply styles
  property: value; - Style rules applied to selected elements
}

Common Properties

color: red; - Text color
background-color: #333; - Background color
font-family: Arial, sans-serif; - Font family
font-size: 16px; - Font size
font-weight: bold; - Font weight (bold, normal, lighter)
text-align: center; - Text alignment (left, right, center)

margin: 0 auto; - Margin (top and bottom 0, left and right auto)
padding: 0; - Padding (space inside element)

border: 1px solid black; - Border (width, style, color)
border-radius: 5px; - Rounded corners

display: flex; - Display type (flex for flexible box layout)
justify-content: center; - Align flex items horizontally

width: 100%; - Width of the element
height: auto; - Height of the element

transition: all 0.3s ease; - Transition effect (property, duration, timing function)
:hover - Pseudo-class for hover effect

@media (max-width: 768px) { } - Media query for responsive design
