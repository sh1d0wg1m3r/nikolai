HTML & CSS Cheatsheet

HTML CSS

HTML

Basic Structure

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    Page content goes here
  </body>
</html>

Common Elements

<h1> to <h6> - Headings
<p> - Paragraph
<a href="URL"> - Link
<img src="image.jpg" alt="Description"> - Image
<ul>, <ol>, <li> - Lists
<table>, <tr>, <th>, <td> - Tables
<br> - Line break
<hr> - Horizontal rule
